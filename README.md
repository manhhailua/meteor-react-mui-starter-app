### Installation

Ensure that you have `git` and `meteor` installed.

```
git clone https://github.com/manhhailua/meteor-react-mui-starter-app.git

cd meteor-react-mui-starter-app/

meteor npm install -g yarn

meteor yarn install

meteor
```
